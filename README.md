# Woolworths Frontend Clone

A frontend clone of the Woolworths website built using HTML, inline CSS, and inline JavaScript for learning and practice purposes.

## Technologies Used
- HTML5
- Inline CSS
- Inline JavaScript

## Features
- Responsive design
- Homepage layout inspired by Woolworths
- Interactive UI
